# CyberTech Expo 2026

Landing page para la feria tecnologica "CyberTech Expo 2026". Presenta informacion clave del evento con un estilo visual moderno y claro.

## Tecnologias utilizadas

- HTML5 semantico
- CSS3 con variables
- Layout con Flexbox y Grid

## Notas

- Diseno responsive para diferentes tamaños de pantalla
- Accesibilidad basica aplicada en la estructura y el formulario

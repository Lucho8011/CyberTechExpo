# Etapa 1: Versión Inicial
- Prompt: "Actúa como un desarrollador Front-end experto. Genera la estructura semántica básica en HTML5 para una landing page de una feria tecnológica llamada 'CyberTech Expo 2026'. El documento debe enlazar un archivo 'style.css'. Incluye: un <header> con el título, un <main> que contenga una sección de descripción general (Hero section con fecha y lugar), y una <section> con el id 'destacados' para uso futuro. Usa etiquetas semánticas estrictas y no incluyas CSS en línea."
- Propósito: Crear el esqueleto HTML inicial.
- Evaluación: Útil.
